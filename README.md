# 图片批量处理工具

基于 React + TypeScript + Vite 构建的图片批量处理应用。

## 功能特性

### 1. 图片智能扩充（Outpainting）
- 智能扩展图片画布
- 在图片四周填充与原图风格一致的内容
- 支持自定义扩充比例（1.25x ~ 2x）

### 2. 图片裂变（Variation）
- 生成风格相同的变体图片
- 保持艺术风格、构图和主题一致
- 支持自定义裂变数量（1-10 张）

### 3. Excel 文生图
- 上传 Excel 文件批量读取提示词
- 每个单元格作为一个独立的提示词
- 支持自定义输出尺寸

### 共有特性
- ✅ 支持上传单张/多张图片
- ✅ 支持选择文件夹批量导入
- ✅ 自定义输出尺寸 + 预设尺寸选项
- ✅ 图片命名前缀 + 自动序号（如 A1, A2, A3...）
- ✅ 批量打包下载 ZIP
- ✅ 三个功能可独立或并行运行

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 8
- **UI**: 自定义 CSS
- **API**: gpt-image-2（通过 https://ai.t8star.cn 中转）

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## API 配置

在应用界面中配置：
- **接口地址**: `https://ai.t8star.cn`（默认）
- **API 密钥**: 你的 gpt-image-2 API 密钥
- **模型**: `gpt-image-2`（默认）

## 输出尺寸

支持以下尺寸：
- 1024x1024（1K 正方形）
- 1024x1536（竖版）
- 1536x1024（横版）
- 1792x1024（横版）
- 1024x1792（竖版）
- 2048x2048（2K 正方形）
- 2048x3072（2K 竖版）
- 3072x2048（2K 横版）
